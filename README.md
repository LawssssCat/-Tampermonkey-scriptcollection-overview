## 安装

打开 `*.user.js` 文件 raw 格式，[tampermonkey](https://www.tampermonkey.net/) 将识别并弹出安装界面

## 原则

1. ✔精简的脚本、✔单一的功能 —— ❌集中的脚本、❌多余的功能
1. 仓库命名方式 `Tampermonkey-scriptcollection-<username>`
